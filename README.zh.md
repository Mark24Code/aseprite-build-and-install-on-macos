# Aseprite build and install on macos

[中文](README.zh.md) | [English](README.md)


在 MacOS 上构建免费的 Aseprite

更新日期: 2025-03-24

支持 Aseprite 1.3.9.1

测试 MacOS 15.3.2
* x86_64 ✅
* arm64 ✅



# 中文说明

## 预先安装

* xcode: https://apps.apple.com/app/xcode/id497799835?mt=12
* brew: https://brew.sh/
* ruby 3+

## 使用

下载项目，命令行输入执行

`./install.rb`

> ⚠️ 注意
> * 中间有个过程会遇到协议界面，只需要输入 q 退出即可继续流程
> * 中间可能需要输入密码，输入回车即可继续流程
